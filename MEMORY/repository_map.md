# Repository Architecture Map — INet-Tool

## Overview

**Package:** INetTool v0.1.1  
**Authors:** Valeria Policastro, Matteo Magnani, Claudia Angelini, Annamaria Carissimo  
**Paper:** Policastro et al. (2024) "INet for network integration" doi:10.1007/s00180-024-01536-8  
**R Version:** >= 3.5.0  
**License:** MIT

The repository implements the INet algorithm for multi-layer network integration. It constructs a **Consensus Network** representing shared information across layers, and **Case-Specific Networks** representing layer-specific information.

## Architectural Layers

### Layer 0: Data Definitions
- `data.R` — loads 4 pre-serialized `.rda` datasets via R's lazy data mechanism

### Layer 1: Input Preparation
- `adj_rename()` — normalizes node names across adjacency matrices (harmonizes heterogeneous input)
- `constructionGraph()` — builds Pearson correlation graphs from raw data matrices, applies percentile threshold

### Layer 2: Distance / Similarity Metrics
- `JWmatrix()` — pairwise weighted Jaccard distance matrix
- `JWmean()` — mean weighted Jaccard distance (scalar)

### Layer 3: Core Algorithm
- `consensusNet()` — the INet iterative consensus algorithm (see algorithm_flow.md)
- Uses `r2r::hashmap` for edge weight and neighbor lookups
- Uses `parallel::clusterApply` for per-graph parallelization

### Layer 4: Post-Processing
- `thresholdNet()` — re-thresholds similar graphs without re-running algorithm
- `densityNet()` — density distribution of mean weights to inform threshold choice
- `specificNet()` — computes case-specific networks via graph set difference

### Layer 5: Visualization
- `plotINet()` — single-layer vs consensus edge comparison plot
- `plotL()` — multi-layer plot via multinet package
- `plotC()` — plot without isolated nodes

### Layer 6: Analysis
- `measuresNet()` — graph-level and node-level measures (transitivity, diameter, modularity, degree, betweenness, hub score, etc.)

## Data Flow (Typical Workflow)

```
Raw data (list of matrices)
    │
    ▼ constructionGraph()
Graphs + Adjacency matrices
    │
    ├──► JWmatrix() / JWmean()   — pre-analysis distance check
    ├──► measuresNet()           — network measures
    ├──► plotL()                 — multi-layer visualization
    │
    ▼ consensusNet()  ◄── adj_rename() (if needed)
Consensus Network + Similar Graphs + Comparison vector
    │
    ├──► thresholdNet()   — re-threshold consensus
    ├──► densityNet()     — help choose threshold
    ├──► specificNet()    — case-specific networks
    ├──► plotINet()       — consensus vs. original comparison
    └──► plotC()          — clean visualization
```
